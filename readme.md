#Installation Instructions
  * Install Ruby 1.9.2
    ** Mac: install via RVM
    ** Windows: RubyInstaller http://rubyinstaller.org/
  * Install RubyGems - http://rubygems.org/pages/download
  * Install Bundler
    ** ```gem install bundler```
  * Install the game's gems 
    ** ```bundle install```
  * Run the game
    ** ```ruby main.rb```
